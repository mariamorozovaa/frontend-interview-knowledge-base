# Operators in JavaScript

## Теория (вопросы и ответы)

---

## Что значат операторы `||=` и `&&=`?

Это логические операторы присваивания (Logical Assignment Operators), появившиеся в ES2021.

Они позволяют объединить проверку условия и присваивание.

---

### Оператор `||=`

Присваивает значение только в том случае, если текущее значение является **falsy**.

```js
let name = "";

name ||= "Anonymous";

console.log(name); // "Anonymous"
```

По сути это сокращение записи:

```js
name = name || "Anonymous";
```

Если значение уже truthy:

```js
let name = "Maria";

name ||= "Anonymous";

console.log(name); // "Maria"
```

---

### Оператор `&&=`

Присваивает значение только тогда, когда текущее значение является **truthy**.

```js
let loggedIn = true;

loggedIn &&= false;

console.log(loggedIn); // false
```

Эквивалент:

```js
loggedIn = loggedIn && false;
```

Если значение было falsy:

```js
let loggedIn = false;

loggedIn &&= true;

console.log(loggedIn); // false
```

Присваивание не произойдет.

---

### Также существует `??=`

Присваивает значение только если переменная равна `null` или `undefined`.

```js
let count = null;

count ??= 10;

console.log(count); // 10
```

---

## Что означают операторы `??` и `||`?

Оба оператора позволяют выбирать значение по умолчанию, но работают по-разному.

---

### Оператор `||`

Возвращает первое **truthy** значение.

```js
const name = "" || "Anonymous";

console.log(name); // "Anonymous"
```

Falsy значения:

```text
false
0
-0
0n
""
null
undefined
NaN
```

Поэтому:

```js
0 || 100;
```

Результат:

```js
100;
```

Иногда это приводит к ошибкам.

---

### Оператор `??`

Возвращает правое значение только если левое равно:

- `null`
- `undefined`

Например:

```js
0 ?? 100;
```

Ответ:

```js
0;
```

Еще пример:

```js
"" ?? "Anonymous";
```

Ответ:

```js
"";
```

---

### Когда использовать?

Если нужно сохранить значения:

- `0`
- `false`
- `""`

используют именно

```js
??
```

---

## Что означает знак `!` перед условием?

Это оператор логического отрицания (Logical NOT).

Он:

1. преобразует значение в boolean;
2. инвертирует его.

Например:

```js
!true; // false

!false; // true
```

---

Еще пример:

```js
const user = null;

if (!user) {
  console.log("Пользователь не найден");
}
```

Поскольку

```js
Boolean(null);
```

равен

```js
false;
```

после отрицания получим

```js
true;
```

---

## Что означает `!!`?

Двойное отрицание используется для явного преобразования значения в boolean.

Например:

```js
!!123; // true

!!0; // false

!!""; // false

!![]; // true

!!{}; // true
```

Эквивалент:

```js
Boolean(value);
```

---

## Что означает `!` после свойства объекта? (`obj!.property`)

Это **не JavaScript**, а **TypeScript**.

Называется:

```text
Non-null Assertion Operator
```

Он сообщает компилятору:

> "Я уверен, что здесь не будет `null` или `undefined`."

Например:

```ts
const input = document.querySelector("input");

input!.focus();
```

Без `!` TypeScript выдаст ошибку:

```text
Object is possibly null.
```

Важно:

Во время выполнения этот оператор никак не влияет на код.

Он существует только для проверки типов.

---

## Что такое Optional Chaining (`?.`)?

Optional Chaining позволяет безопасно обращаться к свойствам объекта.

Без него:

```js
user.address.city;
```

Если

```js
user.address;
```

равен

```js
undefined;
```

получим

```text
TypeError
```

С Optional Chaining:

```js
user?.address?.city;
```

Ответ:

```js
undefined;
```

Ошибка не возникнет.

---

Также работает:

### Для массивов

```js
users?.[0];
```

---

### Для функций

```js
callback?.();
```

Если функция отсутствует — ничего не произойдет.

---

## Что такое бинарные и унарные операторы?

### Унарные операторы

Работают с одним операндом.

Например:

```js
!value;

typeof value;

delete obj.name;

+value;

-value;

++i;

--i;
```

---

### Бинарные операторы

Работают с двумя операндами.

Например:

```js
a + b;

a - b;

a * b;

a / b;

a == b;

a === b;

a && b;

a || b;
```

---

## Что такое оператор `in`?

Проверяет наличие свойства в объекте.

```js
const user = {
  name: "Maria",
};

console.log("name" in user);
```

Ответ:

```js
true;
```

---

Также работает с массивами.

```js
const arr = [10, 20];

console.log(0 in arr);
```

Ответ:

```js
true;
```

Потому что существует элемент с индексом 0.

Важно:

`in` проверяет наличие свойства, а не его значение.

```js
const obj = {
  age: undefined,
};

console.log("age" in obj);
```

Ответ:

```js
true;
```

---

## Что делает оператор `delete`?

Удаляет свойство объекта.

```js
const user = {
  age: 20,
};

delete user.age;

console.log(user);
```

Ответ:

```js
{
}
```

---

Для массива:

```js
const arr = [1, 2, 3];

delete arr[1];

console.log(arr);
```

Ответ:

```js
[1, empty, 3];
```

Длина массива останется прежней.

Поэтому для массивов лучше использовать

```js
splice();
```

---

## Как работают операторы `++` и `--`?

Это операторы инкремента и декремента.

---

### Постфиксная форма

```js
let x = 5;

console.log(x++);
```

Ответ:

```js
5;
```

После этого

```js
console.log(x);
```

Ответ:

```js
6;
```

Сначала возвращается старое значение, затем происходит увеличение.

---

### Префиксная форма

```js
let x = 5;

console.log(++x);
```

Ответ:

```js
6;
```

Сначала происходит увеличение, затем возвращается новое значение.

---

То же самое работает для

```js
--
```

## Практические вопросы собеседований

### Что выведет код?

```js
let count = 0;

count ||= 10;

console.log(count);
```

Ответ:

```js
10;
```

---

### Что выведет код?

```js
console.log(0 || 100);
console.log(0 ?? 100);
```

Ответ:

```js
100;
0;
```

---

### Что выведет код?

```js
console.log(!!"Hello");
console.log(!!0);
```

Ответ:

```js
true;
false;
```

---

### Что выведет код?

```js
const user = {};

console.log(user?.address?.city);
```

Ответ:

```js
undefined;
```

---

### Что выведет код?

```js
const obj = {
  age: undefined,
};

console.log("age" in obj);
```

Ответ:

```js
true;
```

---

### Что выведет код?

```js
const arr = [1, 2, 3];

delete arr[1];

console.log(arr.length);
```

Ответ:

```js
3;
```

---

### Что выведет код?

```js
let x = 5;

console.log(x++);
console.log(x);
```

Ответ:

```js
5;
6;
```

---

### Что выведет код?

```js
let x = 5;

console.log(++x);
```

Ответ:

```js
6;
```

---

## Что сказать на собеседовании

> В JavaScript существует множество операторов для работы с логикой, условиями и объектами. `||` выбирает первое truthy значение, а `??` учитывает только `null` и `undefined`. `?.` позволяет безопасно обращаться к свойствам объекта, а `!` и `!!` используются для логического отрицания и преобразования к boolean. Оператор `in` проверяет наличие свойства, `delete` удаляет свойства объекта, а `++` и `--` изменяют значение переменной на единицу. Операторы `||=` и `&&=` позволяют объединить логическую проверку с присваиванием и делают код более лаконичным.
